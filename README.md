# Self Balancing Robot – eYantra Robotics Competition

This project implements a self-balancing robot using PID control for the IIT Bombay eYantra robotics competition.

## Tasks

- Balance a two-wheeled robot
- Navigate obstacle courses
- Pick and place packages

## Implementation

### Simulation

CoppeliaSim simulation using PID-based stabilization.
